## Nasa_photo
### Приложение отображающее "фото дня" с API сайта NASA
### Приложение "Прогноз погоды"
---
### О проекте:
#### - отображает "фото дня" от API NASA
#### - реализовано сохранение фото в память смартфона
---
## :hammer_and_wrench: Испльзованы для разработки:
#### - kotlin
#### - singleActivity + fragments
#### - material components
#### - MVVM
#### - NASA API
#### - retrofit2
#### - kotlin coroutines
#### - coil
#### - permissions: INTERNET, WRITE_EXTERNAL_STORAGE
---
